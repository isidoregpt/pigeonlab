PigeonLab Setup Guide

Everything you need to get PigeonLab running on your Windows 11 computer — step by step, no technical experience required.

⏱ About 30–45 minutes total
💡
Before you begin You will download and install several free programs. Each one is safe, from official sources, and used by researchers and developers worldwide. None of them cost anything.

Step 1

What You Need

Make sure your computer meets these requirements before you start. If anything is missing, don't worry — we'll install everything together.

🖥️

Windows 11 (64-bit)

PigeonLab runs on Windows 11. To check: press Windows + Pause and look for "64-bit operating system".

🎮

NVIDIA GPU

You need an NVIDIA graphics card (like the A6000 or any modern RTX card). This is what makes video processing fast.

💾

20 GB Free Disk Space

For the software, models, and your video files. More is better — pigeon videos add up quickly.

🌐

Internet Connection

You'll need internet to download software and the AI model. After setup, PigeonLab works entirely offline.

🐍

Python 3.12

The programming language PigeonLab is built on. We'll install this in Step 2.

python.org/downloads →
📦

Node.js 18+

Powers the visual interface you see in your browser. We'll install this in Step 3.

nodejs.org →
🖥️
How to open Command Prompt Throughout this guide, you'll need to type commands. To open Command Prompt: press the Windows key, type cmd, then press Enter. A black window will appear — that's your Command Prompt. Keep it open while working through the steps.
Step 2

Install Python 3.12

Python is the engine that runs PigeonLab's brain. Install it exactly as described below — the version number matters.

1

Download Python 3.12

Go to the official Python website and download the installer

▾
⬇️
Download Python 3.12.10 (64-bit) — Direct Link python.org/ftp/python/3.12.10/python-3.12.10-amd64.exe

Click the link above. A file called python-3.12.10-amd64.exe will download to your Downloads folder.

2

Run the installer

⚠️ There is ONE critical checkbox you must tick

▾
⚠️
This step is critical — don't skip it! On the first screen of the installer, there is a checkbox at the bottom that says "Add Python to PATH". You MUST check this box before clicking Install. If you miss it, Python won't work properly.
  1. Double-click the downloaded file to run it
  2. Check the box that says "Add Python 3.12 to PATH"
  3. Click "Install Now"
  4. Wait for it to finish, then click Close
3

Verify the installation

Confirm Python installed correctly by running a quick check

▾

Open a new Command Prompt window (close and reopen it), then paste this command:

Command Prompt 
python --version
✅
You should see: Python 3.12.10 — if you see this, Python is installed correctly.
Step 3

Install Node.js

Node.js powers the visual interface — the buttons, screens, and charts you'll see when using PigeonLab.

1

Download and install Node.js LTS

LTS means "Long Term Support" — the stable, recommended version

▾
⬇️
Download Node.js LTS (Windows 64-bit) nodejs.org/en/download
  1. Click the link above, then click the big green "LTS" button
  2. Run the downloaded .msi installer
  3. Click Next through all the screens — the defaults are fine
  4. Click Finish when done
2

Verify Node.js installed

Open a new Command Prompt and check

▾
Command Prompt 
node --version
✅
You should see something like: v22.x.x — any number 18 or higher is fine.
Step 4

Install Git

Git is a tool that lets you download PigeonLab onto your computer. Think of it as a very smart file copier.

1

Download and install Git for Windows

Use all the default settings — just click Next through everything

▾
⬇️
Download Git for Windows git-scm.com/download/win
  1. Click the link above — the download starts automatically
  2. Run the installer and click Next through every screen
  3. Click Finish when done
ℹ️
There are many screens during the Git install. You do not need to change anything. Just keep clicking Next and then Finish.
Step 5

Install GPU Drivers & CUDA

CUDA is NVIDIA's toolkit that lets PigeonLab use your graphics card to process video at high speed. Without this, processing would be extremely slow.

1

Update your NVIDIA drivers

Make sure you have the latest drivers for your GPU

▾
⬇️
NVIDIA Driver Downloads nvidia.com/en-us/drivers
  1. Click the link above
  2. Select your GPU model (e.g. RTX A6000) and Windows 11 64-bit
  3. Download and run the installer
  4. Use Express Installation — click through to finish
  5. Restart your computer when prompted
2

Install CUDA Toolkit 12.6

The software bridge between PigeonLab and your GPU

▾
⬇️
CUDA Toolkit 12.6 Download developer.nvidia.com/cuda-12-6-0-download-archive
  1. On the download page, select: Windows → x86_64 → 11 → exe (local)
  2. Download the installer (it's about 3 GB)
  3. Run it and choose Express Installation
  4. Restart your computer when it finishes
⚠️
Restart required! After CUDA installs, you must restart your computer before continuing to the next step.
3

Verify CUDA is working

Confirm your GPU is recognized

▾
Command Prompt 
nvidia-smi
✅
You should see a table showing your GPU name (e.g. NVIDIA RTX A6000) and the CUDA version number. If you see this, you're good to continue.
Step 6

Install PyTorch

PyTorch is the AI framework that PigeonLab uses to run the pigeon detection model. Copy and paste the command below exactly as written.

1

Install PyTorch with CUDA 12.6 support

This one command installs everything — it will take a few minutes

▾
📋
Copy the entire command below — click the Copy button, then paste it into Command Prompt (right-click in the window to paste), then press Enter. Do not type it manually.
Command Prompt — paste and press Enter 
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu126

This downloads about 2–3 GB. You'll see a lot of text scrolling by — that's normal. Wait until you see the Command Prompt cursor again (the blinking C:\> line returns).

✅
When it finishes, you'll see "Successfully installed torch..." in green text.
2

Verify PyTorch can see your GPU

Run a quick test to make sure everything is connected

▾
Command Prompt 
python -c "import torch; print(torch.cuda.is_available()); print(torch.cuda.get_device_name(0))"
✅
You should see:
True
NVIDIA RTX A6000 (or your GPU name)
If you see True, PyTorch is working with your GPU.
Step 7

Set Up PigeonLab

Now we download PigeonLab itself and install all its remaining dependencies. Run these commands one at a time, in order.

📂
Choose where to put PigeonLab — We recommend your Documents folder. All the commands below assume you're working from C:\Users\YourName\Documents. To go there, type cd %USERPROFILE%\Documents and press Enter first.
1

Navigate to Documents and download PigeonLab

Run these two commands in order

▾
Command Prompt — run each line separately 
:: Go to your Documents folder
cd %USERPROFILE%\Documents

:: Download PigeonLab
git clone https://github.com/YOUR-ORG/pigeonlab.git

:: Enter the PigeonLab folder
cd pigeonlab
⚠️
Replace the URL above with the actual PigeonLab repository address your lab administrator provides. It will look like https://github.com/labname/pigeonlab.git
2

Install all Python packages

This installs everything PigeonLab's backend needs

▾
Command Prompt 
:: Make sure you are inside the pigeonlab folder first
cd backend
pip install -r requirements.txt

This will install: FastAPI, SQLite tools, image processing libraries, HuggingFace Hub, and everything else PigeonLab needs. It may take 3–5 minutes.

3

Install the frontend interface

This installs the visual components for the web interface

▾
Command Prompt 
:: Go back to the main pigeonlab folder, then into frontend
cd ..\frontend
npm install

You'll see a lot of text scroll by. This is normal. Wait for the cursor to return.

4

Run the environment check

Make sure everything is installed correctly before continuing

▾
Command Prompt 
cd ..\backend
python scripts\setup_check.py

This runs 16 checks on your system. Each item should show a ✅ green checkmark. The only items that may show ❌ at this point are the SAM 3 checks — that's fine, we'll fix those in the next step.

📋
If any other item shows ❌, note which one and check the Troubleshooting section at the bottom of this guide.
Step 8

Set Up SAM 3 (The AI Model)

SAM 3 is the AI model that finds and tracks pigeons in your videos. It's made by Meta (Facebook) and hosted on HuggingFace — a platform for sharing AI models. You'll need to request access before downloading it.

⏳
This step involves a waiting period After requesting access on HuggingFace, you may need to wait a few hours or up to a day for approval. You can continue to Step 9 and load some sample data while you wait — PigeonLab works without the AI model for exploring the interface.
1

Create a free HuggingFace account

HuggingFace is where AI models are shared — you need a free account to download SAM 3

▾
🤗
Create a free HuggingFace account huggingface.co/join
  1. Click the link above and fill in your name, email, and a password
  2. Verify your email address (check your inbox)
  3. You now have a HuggingFace account
2

Request access to SAM 3

The model requires a one-time access approval from Meta

▾
🦾
SAM 3 Model Page — Request Access huggingface.co/facebook/sam3
  1. Make sure you're logged in to HuggingFace
  2. Visit the link above
  3. Click the "Request access" button on the page
  4. Fill in any requested information and submit
  5. You'll receive an email when access is granted (usually within a day)
3

Log in to HuggingFace on your computer

Do this after your access is approved

▾
Command Prompt 
pip install huggingface_hub
huggingface-cli login

The second command will ask for a token. To get your token:

  1. Go to huggingface.co/settings/tokens
  2. Click "New token", give it a name, and click Generate
  3. Copy the token (starts with hf_)
  4. Paste it into the Command Prompt when asked, then press Enter
4

Download the SAM 3 model files

This downloads about 3–4 GB — leave it running and go get a coffee ☕

▾
Command Prompt — run from the pigeonlab\backend folder 
cd %USERPROFILE%\Documents\pigeonlab\backend
python scripts\download_sam3.py

You'll see a progress bar as the files download. This may take 10–30 minutes depending on your internet speed. Do not close the Command Prompt window while it's running.

✅
When it finishes you'll see: "SAM 3 downloaded to data/models/sam3. You can now start PigeonLab."
5

Run the setup check one more time

Everything should now show green checkmarks

▾
Command Prompt 
python scripts\setup_check.py
✅
All 16 checks should now show ✅ — if they do, PigeonLab is fully set up and ready to use!
Step 9

Launch PigeonLab 🎉

You're ready! There are two ways to start PigeonLab — the easy way and the manual way.

⭐

Easy way — Double-click to start

Recommended for daily use

▾
  1. Open File Explorer and navigate to your pigeonlab folder
  2. Double-click the file called start.bat
  3. Two windows will open — one for the backend, one for the frontend
  4. Wait about 10 seconds, then open your browser
  5. Go to: http://localhost:5173
🎉
PigeonLab is running! You should see the PigeonLab home screen in your browser. Bookmark this address for quick access.
2

Manual way — two Command Prompt windows

Use this if the start.bat file doesn't work

▾

Window 1 — Backend (the brains):

Command Prompt — Window 1 
cd %USERPROFILE%\Documents\pigeonlab\backend
uvicorn main:app --reload --port 8000

Window 2 — Frontend (the screen):

Command Prompt — Window 2 (open a second one) 
cd %USERPROFILE%\Documents\pigeonlab\frontend
npm run dev

Then open your browser and go to http://localhost:5173

3

Load sample data (optional but recommended)

See what PigeonLab looks like with data already in it

▾

To see PigeonLab with example pigeons and videos already loaded, run this in a Command Prompt while PigeonLab is running:

Command Prompt 
cd %USERPROFILE%\Documents\pigeonlab\backend
python seed_data.py

This loads 4 sample pigeons (Alpha, Beta, Gamma, Delta), sample videos, and example analysis results so you can explore all the features.

Step 10

How to Use PigeonLab

Here's how to do the most common tasks. Everything is designed to be self-explanatory, but these guides will get you started quickly.

📹

Process Your First Video

PigeonLab will find, track, and analyze all the pigeons automatically.

  • 1 Click Videos in the left sidebar
  • 2 Click the + Add Videos button
  • 3 Type in the path to your video file (e.g. C:\Videos\session01.mp4)
  • 4 Select the camera type (Overhead, Side, etc.)
  • 5 Enter how many pigeons are in the video
  • 6 Click Process Videos and wait — a progress bar will appear
🐦

Confirm Pigeon Identities

Tell PigeonLab which pigeon is which so it can track them across sessions.

  • 1 On the Home screen, look for "Needs Your Attention"
  • 2 Click Review Now next to "unconfirmed identities"
  • 3 You'll see a pigeon highlighted in the video frame
  • 4 Click the matching pigeon card (Alpha, Beta, etc.)
  • 5 Repeat for each pigeon — use Skip if you're unsure
⚡

Review Quality Flags

When the system spots something unusual, it flags it for your review.

  • 1 On the Home screen, click Check Frames under "Needs Your Attention"
  • 2 You'll see the flagged video frame with a description
  • 3 Click Looks Fine if the system made a mistake
  • 4 Click Fix This to go to the video and correct it manually
🗺️

View Heatmaps & Insights

See where your pigeons spend their time and how they interact.

  • 1 Click Insights in the left sidebar
  • 2 Use the time period selector to choose a date range
  • 3 Click a pigeon's name to filter the heatmap to just that pigeon
  • 4 Scroll down to see the Social Map (who spends time near whom)
  • 5 Click Export Data as CSV to download your data
🐦

Register a New Pigeon

Add a new pigeon to PigeonLab before processing videos that include it.

  • 1 Click Pigeons in the left sidebar
  • 2 Click + Register New in the top right
  • 3 Enter the pigeon's name and physical description (leg bands, markings)
  • 4 Click Save — the pigeon is now registered
📊

Export Your Data

Download your research data as spreadsheet-ready CSV files.

  • 1 Click Insights in the left sidebar
  • 2 Scroll to the bottom of the page
  • 3 Click Export Data as CSV
  • 4 Choose what to include: positions, behaviors, social data
  • 5 Click Export — files download to your computer
📽️

Watch a Video with Overlays

Watch any processed video with colored outlines around each pigeon.

  • 1 Click Videos in the left sidebar
  • 2 Find your video and click Watch ▶
  • 3 Use the slider at the bottom to scrub through frames
  • 4 Press ← → arrow keys to step frame by frame
  • 5 The pigeon cards below the video show who is where
🏋️

Train a Behavior Model

Teach PigeonLab to recognize specific behaviors like feeding or preening.

  • 1 Click Training in the left sidebar
  • 2 Go to Label Clips and label at least 20 clips per behavior
  • 3 Once ready, click Train Model
  • 4 Configure epochs and backbone, then click Launch Training
  • 5 When done, click Set as Active to use the new model
Step 11

Troubleshooting

Something not working? Find your problem below and follow the fix.

What you see What it means & how to fix it
python is not recognized Python wasn't added to PATH during installation. Uninstall Python and reinstall it, making sure to check "Add Python to PATH" on the first screen.
npm is not recognized Node.js didn't install correctly. Close Command Prompt, reopen it, and try again. If still broken, reinstall Node.js.
torch.cuda.is_available() returns False PyTorch can't find your GPU. Make sure CUDA 12.6 is installed and your NVIDIA drivers are up to date. Then reinstall PyTorch using the command in Step 6.
Browser shows "This site can't be reached" PigeonLab isn't running yet. Make sure both Command Prompt windows are open and running (the backend and frontend). Check that you see "Application startup complete" in the backend window.
Access denied when downloading SAM 3 Your HuggingFace access request hasn't been approved yet, or you haven't logged in. Run huggingface-cli login again and check your email for an approval message.
Processing a video stays stuck at "Queued" SAM 3 model files may not be downloaded. Run python scripts\setup_check.py to check. If the checkpoint shows ❌, re-run python scripts\download_sam3.py.
The page is blank or shows an error Open the browser's developer tools (press F12), click Console, and look for red error messages. The most common cause is the backend not running — check the first Command Prompt window.
ModuleNotFoundError for any package A package didn't install properly. Run pip install -r requirements.txt again from the pigeonlab\backend folder.
Heatmap or charts are blank No data has been processed yet, or you need to load sample data. Run python seed_data.py from the backend folder to load example data.
nvidia-smi is not recognized Your NVIDIA drivers aren't installed or need updating. Download the latest driver from nvidia.com/drivers and restart your computer.
Step 12

Daily Startup

Once PigeonLab is set up, starting it each day is quick and easy.

⭐
The fastest way to start PigeonLab Double-click start.bat in your pigeonlab folder. That's it. Wait 10 seconds, then go to http://localhost:5173 in your browser.
🔖
Pro tip: bookmark the URL In your browser, bookmark http://localhost:5173 as "PigeonLab". Every morning you can just double-click start.bat, then click your bookmark.
🛑
To stop PigeonLab Close both black Command Prompt windows. Your data is automatically saved to the database — you won't lose anything.
