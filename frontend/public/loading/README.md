# PigeonLab Loading Images

Copy PNG, JPG, JPEG, WEBP, or GIF files into this folder.

The next time you run `start.bat`, PigeonLab will rebuild `manifest.json` and
show these images in the startup loading screen.

Image tips:

- 16:9 images work best.
- Keep file names friendly, for example `pigeonlab-robot-01.png`.
- The app displays the whole image without cropping, so title text in the image
  will stay visible.
